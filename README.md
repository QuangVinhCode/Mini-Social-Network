Mini Social Network Website

Mô tả dự án:Mini Social Network là một website mô phỏng các chức năng cơ bản của mạng xã hội, cho phép người dùng đăng bài, tương tác và chat realtime.
Dự án tập trung vào xử lý backend, realtime communication và authentication.

Tính năng chính

Đăng ký / đăng nhập người dùng (JWT)

Đăng bài viết, xem bài viết

Like / comment bài viết

Chat realtime giữa người dùng

Cập nhật dữ liệu realtime bằng Socket.IO

Công nghệ sử dụng:

 Backend:Node.js(Express.js),Socket.IO,JWT Authentication

 Frontend:ReactJS,Redux,Ant Design,Socket.io-client

 Database:MongoDB
